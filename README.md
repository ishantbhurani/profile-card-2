## Run Locally

Clone the project

```bash
  git clone git@github.com:ishantbhurani/profile-card-2.git
```

Go to the project directory

```bash
  cd profile-card-2
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
